# Blog Platform Plan

## Overview

Astro-based blog frontend for thamimagi.dev, consuming content from Directus CMS.

## Goals

1. Fast, SEO-optimized static site
2. Consistent UI via @thamimagi/design-system
3. Automated content updates from Directus
4. Excellent Lighthouse scores (90+)

## Current State

- Astro monorepo structure established
- Design system integration complete
- K8s deployment configured
- GitHub Actions CI/CD in place
